# Mini JS Projects

A collection of small vanilla JavaScript builds, each one targeting a specific concept. Built while learning the fundamentals of DOM manipulation, event handling, and JS logic.

> **Live → [Rock Paper Scissors](https://auritrodeykirty07.github.io/mini-js-games/rock-paper-scissors/) · [Digital Clock](https://auritrodeykirty07.github.io/mini-js-games/digital-clock/)**

---

## Projects

### Rock Paper Scissors
Player vs computer with randomised AI picks, score tracking across rounds, and win/lose/draw state handling.

**Concepts practiced:** event listeners, conditional logic, state management, DOM updates

---

### Digital Clock
Live clock that renders the current time and updates every second using `setInterval`.

**Concepts practiced:** `Date` object, `setInterval`, real-time DOM rendering, string formatting

---

## Why this repo exists

Every concept here (event delegation, state tracking, real-time rendering) show up in much larger projects. These were deliberate practice, not just tutorials copy-pasted and forgotten.
